如何搭建个人网站

  • 内容
  • 相关

有很多人也想搭建一个网站，却不知道从什么地方入手，就开始百度搜索什么免费建站，一键建站。这些东西基本上说都是。。。而且一个比一个坑，就拿我小学用过的凡科建站说吧：

1.一个月登陆一次，不然就关站。

2.不许绑定域名

3.文件上传有限制

简直就是坑钱来的。

废话不说了，教大家搭建主机

方案一：不费力气，不费钱

此方法适合懒人~~

此方法很简单，去 blog.163.com 或 blog.sina.com 申请个博客就行了，比免费建站要好得多。一分钱都不费

优点：

1.网站不会出现宏机

2.省力气，省钱

缺点：

1.不能绑定独立域名

2.有广告（自己也不赚钱）


方案二：需要一丢丢的技术，省钱

此方法也省钱，就是不省力气

首先，我们需要去freenom去申请一个免费的tk域名（也可以去注册一个收费域名）

网址：www.freenom.com

然后，去找一个免费的dns（因为freenom的dns服务器不太靠谱）

网址：https://www.cloudflare.com/

如果不懂英文，可以使用百度云加速 su.baidu.com

最后去找一个免费空间，在百度内搜索“免费空间”或“免费主机”

这里我不推荐，也可以用我们旗下的小徐云网络 idc.smallxu.cc

最后到了选择博客系统的时候，博主用的是emlog，轻量化，可以附加模板

这里附上这几个博客系统：

1.emlog 轻量级，不过最近官网貌似不能访问，地址：www.emlog.net 想下载的话可以百度emlog5.3.1，emlog的6.0版本有很多bug，建议不要用

2.wordpress 国外著名博客系统，因为谷歌被墙和它使用谷歌字体，所以国内访问可能很慢（换个前端主题就OK了） 地址：https://cn.wordpress.org/

3.zblog 此系统最初开发的是asp系统，但是近几年开发出了php版本。地址：www.zblogcn.com

4.typecho php系统，作者也没用过，不知道怎么样
